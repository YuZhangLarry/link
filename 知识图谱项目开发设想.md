# 第一部分：开发前必读

## 项目概述

link 是一个基于知识图谱和 RAG（Retrieval-Augmented Generation）的智能知识管理系统。项目采用微服务架构，集成了向量检索、图检索、联网搜索等功能，提供全方位的知识管理和智能问答服务。

### 技术栈总览

| 类别 | 技术 | 版本 | 用途 |
|------|------|------|------|
| **后端** | Go | 1.25.6 | 主要开发语言 |
| **前端** | Vue 3 | 3.x | 用户界面 |
| **Python** | Python | 3.9+ | 文档解析服务 |(二期)
| **图数据库** | Neo4j | 5.15.0 | 知识图谱存储 |
| **关系数据库** | MySQL | 8.0+ | 元数据存储 |
| **向量数据库** | Milvus | 2.3+ | 语义检索 |
| **缓存** | Redis | 7.0+ | 性能优化 |
| **框架** | Eino | latest | RAG 应用框架 |

### 2. Agent 智能体模块 (internal/agent/)

**位置**: `internal/agent/`

#### 功能特性
- **ReACT 框架**: Reasoning + Acting 循环
- **工具集成**: 调用多种工具完成任务
- **知识推理**: 基于图谱推理
- **多轮对话**: 维护对话上下文

#### 模块结构
```
internal/agent/
├── agent.go         # Agent 核心逻辑
├── tools.go         # 工具集成和注册
└── react.go         # ReACT 框架实现
```

#### ReACT 工作流程
```
用户输入
  ↓
观察 (Observe)
  - 获取当前状态
  - 分析可用信息
  ↓
思考 (Reason)
  - 理解问题意图
  - 决定需要做什么
  - 选择合适的工具
  ↓
行动 (Act)
  - 执行工具调用
  - 获取执行结果
  ↓
观察结果 → 循环直到完成任务
  ↓
返回最终答案
```

#### 可用工具
```go
// 工具列表
type Tool interface {
    Name() string
    Description() string
    Execute(ctx context.Context, input string) (string, error)
}

// 内置工具
- WebSearch      // 联网搜索
- GraphQuery     // 知识图谱查询
- VectorSearch   // 向量检索
- DatabaseQuery  // 数据库查询
- DocumentRead   // 文档读取
```

#### 开发要点
- **工具注册**: 在 `tools.go` 中注册新工具
- **推理限制**: 最多执行 N 步（防止死循环）
- **错误处理**: 工具执行失败时回退到其他工具
- **日志记录**: 记录每步的观察、思考、行动

### 3. 检索服务模块 (internal/searchutil/)

**位置**: `internal/searchutil/`

#### 功能特性
- **混合检索**: BM25 + 向量检索
- **多路召回**: 同时使用多种策略
- **重排序**: 对结果精排
- **GraphRAG**: 图谱增强检索

#### 模块结构
```
internal/searchutil/
├── retriever.go     # 统一检索接口
├── bm25.go          # BM25 关键词检索
└── vector.go        # 向量语义检索
```

#### 检索流程
```
用户查询
  ↓
查询预处理
  - 分词
  - 停用词过滤
  - 同义词扩展
  ↓
多路召回
  ├─ BM25 检索 (关键词匹配)
  ├─ 向量检索 (语义相似度)
  ├─ 图谱检索 (关系扩展)
  └─ 联网搜索 (实时信息)
  ↓
结果合并
  - 去重
  - 权重融合
  ↓
重排序 (Rerank)
  - 相关性评分
  - 质量评估
  ↓
返回 Top-K 结果
```

#### 检索策略
```go
// 召回配置
type RetrievalConfig struct {
    BM25Ratio      float64  // BM25 权重: 0.3
    VectorRatio    float64  // 向量权重: 0.5
    GraphRatio     float64  // 图谱权重: 0.2
    TopK           int      // 返回数量: 10
    Rerank         bool     // 是否重排序: true
}

// 使用示例
results := retriever.Retrieve(ctx, query, config)
```

#### 开发要点
- **索引构建**: 定期重建 BM25 索引
- **向量预热**: 启动时预热 Milvus 连接
- **缓存策略**: 热点查询缓存 1 小时
- **性能优化**:

  - 批量查询

### Redis - 缓存

**用途**: 缓存热点数据、会话管理

#### 缓存策略
```go
// 查询结果缓存
func (r *Retriever) Retrieve(ctx context.Context, query string) ([]Result, error) {
    // 1. 尝试从缓存获取
    cacheKey := fmt.Sprintf("search:%s", query)
    cached, _ := r.redis.Get(ctx, cacheKey)
    if cached != nil {
        return cached, nil
    }

    // 2. 执行检索
    results := r.doRetrieve(ctx, query)

    // 3. 写入缓存 (TTL: 1h)
    r.redis.Set(ctx, cacheKey, results, 1*time.Hour)

    return results, nil
}
```

#### 缓存配置
```yaml
cache:
  # 查询结果缓存
  query:
    ttl: 3600        # 1 hour
    max_size: 10000

  # 用户会话
  session:
    ttl: 86400       # 24 hours

  # 向量检索缓存
  vector:
```

### 配置加载

```go
// internal/config/loader.go
package config

import (
    "fmt"
    "os"

    "gopkg.in/yaml.v3"
)

type Config struct {
    Server   ServerConfig   `yaml:"test-gin-base"`
    Database DatabaseConfig `yaml:"database"`
    LLM      LLMConfig      `yaml:"llm"`
    // ...
}

func Load(path string) (*Config, error) {
    data, err := os.ReadFile(path)
    if err != nil {
        return nil, err
    }

    var cfg Config
    if err := yaml.Unmarshal(data, &cfg); err != nil {
        return nil, err
    }

    // 环境变量覆盖
    if port := os.Getenv("SERVER_PORT"); port != "" {
        cfg.Server.Port = port
    }

    return &cfg, nil
}
```

```go
    "github.com/neo4j/neo4j-go-driver/v5/neo4j"
)

type Neo4jClient struct {
    driver neo4j.DriverWithContext
}

func NewNeo4jClient(uri, username, password string) (*Neo4jClient, error) {
    driver, err := neo4j.NewDriverWithContext(
        uri,
        neo4j.BasicAuth(username, password, ""),
    )
    if err != nil {
        return nil, err
    }
    return &Neo4jClient{driver: driver}, nil
}

func (c *Neo4jClient) Query(ctx context.Context, cypher string, params map[string]any) (neo4j.ResultWithContext, error) {
    session := c.driver.NewSession(ctx, neo4j.SessionConfig{AccessMode: neo4j.AccessModeRead})
    defer session.Close(ctx)

    return session.Run(ctx, cypher, params)
}

func (c *Neo4jClient) Execute(ctx context.Context, cypher string, params map[string]any) error {
    session := c.driver.NewSession(ctx, neo4j.SessionConfig{AccessMode: neo4j.AccessModeWrite})
    defer session.Close(ctx)

    _, err := session.Run(ctx, cypher, params)
    return err
}
```

```go
// internal/database/mysql.go
package database

import (
    "database/sql"
    "gorm.io/driver/mysql"
    "gorm.io/gorm"
)

type MySQLClient struct {
    db *gorm.DB
}

func NewMySQLClient(host, port, database, username, password string) (*MySQLClient, error) {
    dsn := fmt.Sprintf("%s:%s@tcp(%s:%s)/%s?charset=utf8mb4&parseTime=True",
        username, password, host, port, database)

    db, err := gorm.Open(mysql.Open(dsn), &gorm.Config{})
    if err != nil {
        return nil, err
    }

    return &MySQLClient{db: db}, nil
}

func (c *MySQLClient) Create(value interface{}) error {
    return c.db.Create(value).Error
}

func (c *MySQLClient) Find(dest interface{}, conds ...interface{}) error {
    return c.db.Find(dest, conds...).Error
}

func (c *MySQLClient) First(dest interface{}, conds ...interface{}) error {
    return c.db.First(dest, conds...).Error
}
```
